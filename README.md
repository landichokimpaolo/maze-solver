## Maze Solver

# Dependency
- [Pillow](https://github.com/python-pillow/Pillow)

# How to use?
Specify the path of the maze file

```bash
$ python3 maze.py "mazes/maze2.txt"
```

# Creating your own maze
A - Start\
B - End\
\# - Wall\
space - Path

See file mazes/maze1.txt for reference

# Screeshot
